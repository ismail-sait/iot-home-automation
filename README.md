 iot-home-automation :
 IoT Based Home Automation System

 Project Overview:
This project is an IoT-based home automation system using NodeMCU (ESP8266) and relay module. The system allows users to control electrical appliances through Wi-Fi using a mobile phone or web browser.

 Components Used:
- NodeMCU (ESP8266)
- Relay Module
- Bulb / Fan
- Jumper Wires

 Technologies Used:
- Embedded C
- Arduino IDE
- Wi-Fi
- IoT

Working Principle:
The NodeMCU connects to Wi-Fi and creates a web server. The user can control the relay ON/OFF through a web page, which controls the electrical appliances.

 Applications:
- Smart Home
- IoT Automation
- Remote Appliance Control
